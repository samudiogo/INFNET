Elabore uma classe chamada FraseAleatoria. Esta classe representa uma frase aleatória entre 5 e 100 caracteres - esta frase ou suas palavras não precisam fazer sentido. Deve possuir dois construtores, um construtor default que gera uma frase de 25 caracteres e um outro construtor que recebe um inteiro com o tamanho da String. Os construtores devem ser elaborados de forma a minimizar o código escrito. A frase aleatória deve ficar em um atributo privado da classe.

Desenvolva um método B que retorna o número de vezes que uma determinada palavra, passada por parâmetro, aparece na frase aleatória.

Desenvolva um método C que retorne um ArrayList contendo todas as palavras da frase aleatória que são maiores que um certo tamanho, passado por parâmetro. *Quando demonstrar o uso deste método na main, imprima no console todas as palavras do ArrayList que se repetem, junto com o número de repetições*

Desenvolva um método D que recebe um caractere como parâmetro e substitui todas suas ocorrências na frase aleatória pelo caractere '?'.
